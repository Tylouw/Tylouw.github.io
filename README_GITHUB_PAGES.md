# Robotics Portfolio Website — GitHub Pages Setup

This is a static website made for GitHub Pages.

## Files
- `index.html` — main page
- `styles.css` — styling
- `script.js` — modal behavior for “More information”
- `assets/placeholders/` — replace these placeholder images with your own screenshots and thumbnails

## Before publishing
1. Open `index.html`
2. Replace:
   - `yourusername`
   - `your.email@example.com`
   - `your-linkedin`
3. Replace placeholder images in `assets/placeholders/`
4. Replace the project text with your final wording
5. Optional: replace the large placeholder images with real `<video>` embeds or YouTube iframes
