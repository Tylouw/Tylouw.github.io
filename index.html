Brobot — Custom 6-DoF Robot Arm

Designed and built a complete 6-DoF robot arm from scratch. Developed the mechanical structure in CAD, produced parts by 3D printing, designed the electronics architecture, and implemented a custom C++ controller on ESP32. Implemented forward/inverse kinematics, coordinated axis motion, Jacobian-based Cartesian jogging, and safety logic.

Whiteboard Drawing Robot

Developed a whiteboard drawing system using a UR robot, ROS 2, and OptiTrack, with custom calibration and robot integration for accurate drawing tasks. Self derived the hand-eye calibration to find transformation from endeffector to marker and robot base to Optitrack world origin using Ax=b. Due to continuous optical tracking, the system is able to adapt to changing whiteboard and pencil holder position during drawing process.
